16F15325 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// External Oscillator mode selection bits : EC above 8MHz; PFM set to high power
// Power-up default value for COSC bits : EXTOSC operating per FEXTOSC bits
// Clock Out Enable bit : CLKOUT function is disabled; i/o or oscillator function on OSC2
// Clock Switch Enable bit : Writing to NOSC and NDIV is allowed
// Fail-Safe Clock Monitor Enable bit : FSCM timer enabled
#pragma config FEXTOSC = ECH, RSTOSC = EXT1X, CLKOUTEN = OFF, CSWEN = ON, FCMEN = ON
#pragma config <setting>=<literal constant>
For example:
// External Oscillator mode selection bits : EC above 8MHz; PFM set to high power
// Power-up default value for COSC bits : EXTOSC operating per FEXTOSC bits
// Clock Out Enable bit : CLKOUT function is disabled; i/o or oscillator function on OSC2
// Clock Switch Enable bit : Writing to NOSC and NDIV is allowed
// Fail-Safe Clock Monitor Enable bit : FSCM timer enabled
#pragma config FEXTOSC = 0x7, RSTOSC = 0x7, CLKOUTEN = 0x1, CSWEN = 0x1, FCMEN = 0x1
#pragma config <register>=<literal constant>
For example:
// External Oscillator mode selection bits : EC above 8MHz; PFM set to high power
// Power-up default value for COSC bits : EXTOSC operating per FEXTOSC bits
// Clock Out Enable bit : CLKOUT function is disabled; i/o or oscillator function on OSC2
// Clock Switch Enable bit : Writing to NOSC and NDIV is allowed
// Fail-Safe Clock Monitor Enable bit : FSCM timer enabled
#pragma config CONFIG1 = 0x3FFF
For example:
// IDLOC @ 0x8000
#pragma config IDLOC0 = 0x16383

Configuration Registers & Settings

Register: CONFIG1 @ 0x8007
FEXTOSC External Oscillator mode selection bits
ECH EC above 8MHz; PFM set to high power
ECM EC for 500kHz to 8MHz; PFM set to medium power
ECL EC below 500kHz; PFM set to low power
OFF Oscillator not enabled
HS HS (crystal oscillator) above 4MHz; PFM set to high power
XT XT (crystal oscillator) above 500kHz, below 4MHz; PFM set to medium power
LP LP (crystal oscillator) optimized for 32.768kHz; PFM set to low power
RSTOSC Power-up default value for COSC bits
EXT1X EXTOSC operating per FEXTOSC bits
HFINT1 HFINTOSC (1MHz)
LFINT LFINTOSC
SOSC SOSC
EXT4X EXTOSC with 4x PLL, with EXTOSC operating per FEXTOSC bits
HFINTPLL HFINTOSC with 2x PLL, with OSCFRQ = 16 MHz and CDIV = 1:1 (FOSC = 32 MHz)
HFINT32 HFINTOSC with OSCFRQ= 32 MHz and CDIV = 1:1
CLKOUTEN Clock Out Enable bit
ON CLKOUT function is enabled; FOSC/4 clock appears at OSC2
OFF CLKOUT function is disabled; i/o or oscillator function on OSC2
CSWEN Clock Switch Enable bit
ON Writing to NOSC and NDIV is allowed
OFF The NOSC and NDIV bits cannot be changed by user software
FCMEN Fail-Safe Clock Monitor Enable bit
ON FSCM timer enabled
OFF FSCM timer disabled
Register: CONFIG2 @ 0x8008
MCLRE Master Clear Enable bit
ON MCLR pin is Master Clear function
OFF MCLR pin function is port defined function
PWRTE Power-up Timer Enable bit
OFF PWRT disabled
ON PWRT enabled
LPBOREN Low-Power BOR enable bit
OFF ULPBOR disabled
ON ULPBOR enabled
BOREN Brown-out reset enable bits
ON Brown-out Reset Enabled, SBOREN bit is ignored
NSLEEP Brown-out Reset enabled while running, disabled in sleep; SBOREN is ignored
SBOREN Brown-out reset enabled according to SBOREN bit
OFF Brown-out reset disabled
BORV Brown-out Reset Voltage Selection
LO Brown-out Reset Voltage (VBOR) set to 1.9V on LF, and 2.45V on F Devices
HI Brown-out Reset Voltage (VBOR) is set to 2.7V
ZCD Zero-cross detect disable
OFF Zero-cross detect circuit is disabled at POR.
ON Zero-cross detect circuit is always enabled
PPS1WAY Peripheral Pin Select one-way control
ON The PPSLOCK bit can be cleared and set only once in software
OFF The PPSLOCK bit can be set and cleared repeatedly by software
STVREN Stack Overflow/Underflow Reset Enable bit
ON Stack Overflow or Underflow will cause a reset
OFF Stack Overflow or Underflow will not cause a reset
